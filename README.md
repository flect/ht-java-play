ht-java-play
============

Sample project that use Play 2 Java for Heroku training.

How to deploy
--------------

### Step1  

Clone app from Github  

```
$ git clone git@github.com:flect/ht-java-play.git
$ cd ht-java-play/
```

### Step2  

Create Heroku application

```
$ heroku create
```

### Step3  

Deploy application
```
$ git push heroku master
```

### Step 4

Visit application

```
$ heroku open
```
